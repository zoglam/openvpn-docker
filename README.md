за иксами адрес вбить компа в локальной сети


Запустить сервер
```sh
make build && make run
```

Что выпустить серт
```sh
# сначала скачать ca серт, ключ и ta
make copy
# а после выполнить выписывание и сборку ovpn. CLIENT имя вбить прям в add.sh
# на 4 строке
make add
```
